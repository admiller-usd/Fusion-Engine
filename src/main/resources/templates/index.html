Welcome to the Fusion Engine!

This is a web service that accepts RESTful interactions to process coordinate metadata.
You can submit new coordinates via the submit endpoint, or query for data via the query endpoint.

How to submit coordinates

Must POST to the endpoint /submit with two parameters: latitude and longitude
Example: /submit?latitude=37.42512&longitude=-122.08334
The json response body will return a UUID which can be used to query for results.

How to query for coordinates

Must GET to the endpoint /query with one parameter: uuid
Example: /submit?uuid=0793821e-0534-43a2-9b62-456477c924e6
The json response body will return your corresponding coordinate pair.
